While i was doing Burp's lab for deserialization, the extension for deserialization stopped working. So i thought that why not create my own simple tool to automate it.

Before using, change 
1) Path of java
2) Path of ysoserial
3) User creds.

Run the script with

```
python3 SDS.py

```
